#  Consumo de Energia

##  Objetivo
Este projeto calcula o consumo mensal de energia de um aparelho elétrico.

##  Linguagem utilizada
Python

##  Fórmula utilizada

consumoMensal = (potencia * horasDia * 30) / 1000

##  Como executar

Execute o comando:

python app.py
